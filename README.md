# Rain-Prediction
I developed a Rain Prediction Model using Logistic Regression to predict whether it will rain or not based on historical weather data. The problem was formulated as a binary classification task, where the output is either rain or no rain.

The project involved data preprocessing, including handling missing values, encoding categorical variables, and applying feature scaling to improve model performance. I used techniques like train-test split and evaluated the model using accuracy metrics.

Logistic Regression was chosen because of its simplicity, interpretability, and effectiveness for binary classification problems. The model learns the relationship between input features (such as temperature, humidity, wind speed, etc.) and the probability of rainfall.

This project helped me gain hands-on experience in data cleaning, feature engineering, model building, and evaluation, and strengthened my understanding of real-world machine learning workflows.
